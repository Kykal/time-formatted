# What is this?

This package brings functions that convert seconds into formatted time. Use the seconds you want to convert as argument.
```JS
shortTime(int seconds)
```
```JS
longTime(int seconds)
```

# Instalation

Using ``npm``
```SSH
npm i @kykal/time-formatted
```
